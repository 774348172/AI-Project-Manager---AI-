# AI Project Knowledge Base Manager - 知识库项目管家

AI Project Knowledge Base Manager (codename: Bai Ze) - An intelligent AI-powered project management and knowledge base assistant for developers and teams.  
本项目是一款人工智能驱动的「知识库 + 项目管家」工具，依托 AI 能力赋能项目全流程管理，支持任务分配、进度追踪、智能提醒、文档沉淀与团队协作等核心功能。适用于个人开发者、研发团队及 AI 项目运维管理场景。
